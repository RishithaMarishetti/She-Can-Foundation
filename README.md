# She Can Foundation

A simple and responsive NGO website for She Can Foundation built using HTML, CSS, and JavaScript, focused on education and learning opportunities for students and communities.

## Features

* Responsive design
* Clean and minimal UI
* Smooth scrolling
* Interactive buttons
* NGO-focused landing page

## Technologies Used

* HTML
* CSS
* JavaScript

## Purpose

This project was created as a frontend internship task to represent an NGO website focused on education, mentorship, and learning opportunities for students and communities.
